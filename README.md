QR Code Generator

A clean, fast and minimal QR Code Generator built using Qrious.js.

This tool is based on the original work by Marko Denic, with full credit and acknowledgment to his project:
 Original source: https://github.com/markodenic/qr-code-generator
 Website: https://markodenic.com

This version has been adapted and restyled by Luc Constantin, with modern refinements while keeping Marko’s original simplicity and quality.

Features
Generates QR codes instantly using Qrious
Clean, minimalist UI inspired by the original Marko Denic design
Fully responsive
Supports multiple sizes (150px – 400px)
Download as PNG
Auto-updates as you type
Dark mode support, respecting the user’s system preference
Brand-customized theme (Accolades.dev colors)

Custom Styling
This version uses Luc Constantin’s brand colors:
Blue: #3656e8
Green: #33f078
Neutral black/white palette
The layout and spacing remain faithful to the original.

Changes Compared to the Original
- This fork includes:
- Updated styling with new brand colors
- Dark-mode support
- Small UI refinements
- Minor accessibility improvements
- Default URL changed to https://accolades.dev
- Removed logo-in-QR experimental attempt to keep absolute scanning reliability
- Cleaned and modernized CSS
- Updated footer attribution
- All QR generation logic still uses Qrious, exactly as the original does.

Usage
Simply open index.html in any browser and begin typing a URL or text.
You can change the QR size using the dropdown and download the PNG with one click.

No build tools needed.
No dependencies.
100 percent client-side.
